# 🏕️ 沙海草原自驾旅行计划 · 给少剑的专属页面

这是为我们准备的「北京 → 乌兰布统」周末自驾旅行计划网页，包含完整的行程安排、费用预算、路线动线和目的地介绍。你可以直接在页面里打卡心情、给我留言、确认预算。

---

## 🗺️ 行程概览

48小时沉浸式自驾，串联四重地貌：
沙漠穿沙公路 → 高山风车草甸 → 坝上丘陵草原 → 森林景观大道

从北京夜间出发，把通勤时间留给夜晚，周末完整留给风景。

---

## ✨ 页面功能

### 🎯 你可以互动操作
- **当日心情打卡**：从 5 种心情表情中选择，记录当下状态
- **给我留言**：写下想说的话，提交后我会立刻收到邮件提醒
- **预算批准**：一键确认整趟旅行的预算方案
- **旅行方案切换**：支持多套行程方案查看，后续可随时新增路线

### 📖 行程内容展示
- **按天行程时间线**：每日行程按时间排序，点击展开即可查看详细安排、交通方式和备注提示
- **预算明细清单**：按交通、住宿、餐饮、门票分类展示，自动计算分类合计与总费用
- **路线可视化地图**：SVG 绘制全程自驾动线，直观看到整趟行程的走向与途经点位
- **目的地介绍卡片**：每个核心景点配有精简简介，点击卡片可展开查看详细说明
